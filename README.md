# Video Game Sales Analysis

## Description
An exploratory analysis of the best-selling video games of all time, using Python fundamentals to uncover trends in global sales across genres, regions, and publishers.

## Project Structure
| File | Task | Topic |
|------|------|-------|
| task1_variables.py | Task 1 | Variables and Arithmetic |
| task2_strings.py | Task 2 | String Processing |
| task3_lists.py | Task 3 | Lists and Tuples |
| task4_loops.py | Task 4 | Loops and Conditionals |
| task5_dictionaries.py | Task 5 | Dictionaries |
| task6_functions.py | Task 6 | Functions |

## Data Source
Adapted from VGChartz (public domain estimates). The dataset contains 20 of the best-selling video games of all time, including sales figures for North America, Europe, Japan, and worldwide.

## Methods and Tools
- Python (variables, loops, conditionals, lists, dictionaries, functions)

## Key Findings
- Wii Sports is the #1 best-selling game with 82.74M global sales
- Nintendo published 17 out of 20 games in the dataset, dominating the industry
- North America significantly outsold Japan across all games (295.75M vs 78.72M)
- Sports and Platform genres had the highest combined global sales
- 6 games in the dataset were released before the year 2000
